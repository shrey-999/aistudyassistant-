# AI Study Assistant

A simple Python application that uses the Gemini API to perform different study-related tasks.

## Features

- Explain a concept
- Summarize text
- Generate quiz questions
- Handles empty input

## Technologies Used

- Python
- Gemini API
- python-dotenv

## Installation

1. Clone the repository

```bash
git clone <repo-url>
Install dependencies
pip install -r requirements.txt
Create a .env file
GEMINI_API_KEY=your_api_key
Run the application
python app.py
